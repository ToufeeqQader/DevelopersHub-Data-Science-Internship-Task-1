# Task 1: Exploring and Visualizing a Simple Dataset

### 1. The Task Objective
To understand how to read, summarize, and visualize a dataset using the Iris dataset.

### 2. My Approach
* Loaded the dataset using the `pandas` library.
* Inspected the data structure using `.shape`, `.columns`, and `.head()`.
* Conducted Exploratory Data Analysis (EDA) by creating a scatter plot, histogram, and box plot using `matplotlib` and `seaborn`.

### 3. Results and Insights
The visualizations reveal that the Setosa species is the easiest to distinguish from the others, primarily due to its distinct sepal proportions and significantly smaller petal lengths shown in the histogram.

Additionally, the box plot highlights that while Setosa generally has a higher average sepal width, there are a few noticeable outliers present in the Virginica and Versicolor species.
